# ⚽ Futsal Club Manager (フットサル部員管理システム)

フットサル部の「練習日程」と「部員情報」を管理するためのWebアプリケーションです。
Dockerを使用しているため、コマンド1つで誰でも簡単に環境を構築・起動できます。

## 🛠 必要なもの (事前準備)
このシステムを動かすには、以下のソフトウェアがインストールされている必要があります。

* **Docker Desktop** (起動していること)
* **VS Code** (推奨)

## 🚀 起動手順 (How to Start)

### 手順 1: コードのダウンロード
このリポジトリを自分のPCにダウンロード（Clone）し、VS Codeでそのフォルダを開いてください。

### 手順 2: ターミナルを開く
VS Codeの上部メニューから `Terminal` > `New Terminal` をクリックし、画面下にターミナル（黒い画面）を出します。

### 手順 3: 起動コマンドを実行
ターミナルに以下のコマンドを入力して Enter キーを押してください。
初回はダウンロードに数分かかる場合があります。

```bash
docker-compose up --build
```


手順 4: ブラウザでアクセス
ログが流れ、動きが止まったら起動完了です。 以下のURLをクリックしてアクセスしてください。

👉 http://localhost:5000
